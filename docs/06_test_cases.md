# OneDone — Test Cases

Initial MVP flows to test:

1. First-time user:
Register → Onboarding → Starter Access → Home.

2. Cancel subscription:
New Task → Clarification → App Store → Task Result.

3. Starter expired:
Open app → Starter expired gate → Start App Store trial.

4. Copy reply:
Generate reply → Copy → Did you send it? → Waiting for Reply.

5. Reminder:
Create reminder → Local notification → Open exact task.
